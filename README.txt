Computer Society Weekly Meetups (NISB | IEEE)

This project is a simple website showcasing weekly CS meetups with:
- Learning objectives
- Prerequisites
- Materials (dummy PDFs)
- Venue, time, and location
- Google Calendar integration
- Neon-styled UI with a fullscreen background video

NOTE: All PDF files inside server/public/materials/ are DUMMY placeholder files.
They do NOT contain any meaningful content and are included only to demonstrate downloads.

Tech Stack:
Frontend: React + Vite + TailwindCSS v4
Backend: Node.js + Express.js

How to Run:
Backend → cd server && npm install && npm run dev
Frontend → cd client && npm install && npm run dev
